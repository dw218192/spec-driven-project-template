# ADRs

Architecture Decision Records, numbered `docs/adr/<n>-<slug>.md`. Author with the
**`adr`** skill (template: `.claude/skills/adr/references/adr-template.md`). Accepted
ADRs are immutable — supersede additively (a new ADR with `supersedes: [ADR-N]`);
reverse links are derived by the skill's `adr_status.py`.
